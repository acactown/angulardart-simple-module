# angulardart_simple_module

A library for Dart developers. It is awesome.

## Usage

A simple usage example:

    import 'package:angulardart_simple_module/recipe_book_module.dart';

    main() {
      applicationFactory()
        .addModule(new RecipeBookModule())
        .run();
    }
